# Emmet

## ID and CLASS attributes

```
#header
<div id="header"></div>

.title
<div class="title"></div>

form#search.wide
<form id="search" class="wide"></form>

p.class1.class2.class3
<p class="class1 class2 class3"></p>
```

## Child: >

```
nav>ul>li

<nav>
    <ul>
        <li></li>
    </ul>
</nav>
```

## Multiplication: \*

```
ul>li*5

<ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>
```
